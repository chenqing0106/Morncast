# 本地预览扫码指南

手机扫码体验 Morncast Demo 的完整步骤。

---

## 前提条件

- 手机与 Mac 在**同一 WiFi** 下
- 本机局域网 IP：`192.168.1.123`（若换网络需重新查询：`ipconfig getifaddr en0`）

---

## 一键启动（推荐）

在终端执行以下命令，自动启动服务器并打印二维码：

```bash
cd /Users/qingchen/Personal-Learning-Vault/项目/北理黑客松/morncast-demo \
  && python3 -m http.server 8080 & \
  node -e "const q=require('/Users/qingchen/.nvm/versions/node/v24.14.0/lib/node_modules/npm/node_modules/qrcode-terminal/lib/main.js');q.generate('http://192.168.1.123:8080',{small:true},c=>console.log(c))"
```

用手机扫描终端里打印的二维码即可访问。

## 分步操作

### 第一步：启动 HTTP 服务器

```bash
cd /Users/qingchen/Personal-Learning-Vault/项目/北理黑客松/morncast-demo
python3 -m http.server 8080
```

### 第二步：生成二维码

新开一个终端窗口，执行：

```bash
node -e "
const q = require('/Users/qingchen/.nvm/versions/node/v24.14.0/lib/node_modules/npm/node_modules/qrcode-terminal/lib/main.js');
q.generate('http://192.168.1.123:8080', {small: true}, c => console.log(c));
"
```

### 第三步：手机扫码

- **iOS**：打开相机直接对准扫描
- **Android**：微信扫一扫 / 系统相机

访问地址：`http://192.168.1.123:8080`

## 二维码有效期说明

二维码编码的是局域网 IP，**不是固定的**，以下情况会失效：

- 换了 WiFi 网络（IP 段不同）
- 路由器重新分配了 IP（DHCP 租约到期）
- Mac 重启后重新连接，IP 有时会变

每次使用前确认 IP 是否变化：

```bash
ipconfig getifaddr en0
```

若 IP 变了，重新执行一键启动命令即可生成新二维码。

### 想要固定链接？

| 方案 | 适用场景 | 操作 |
|------|----------|------|
| **Netlify Drop** | 任意网络可访问，永久地址 | 拖拽文件夹到 drop.netlify.com |
| **GitHub Pages** | 有 GitHub 仓库时 | 开启 Pages，自动生成 HTTPS 地址 |
| **固定本机 IP** | 只在同一 WiFi 使用 | Mac 系统设置 → 网络 → WiFi → 详细信息 → TCP/IP → 手动配置 |

## 关闭服务器

```bash
# 查找端口占用进程
lsof -ti:8080

# 终止服务器
kill $(lsof -ti:8080)
```
